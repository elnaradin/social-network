# env variables:


```
EMAIL_FROM=;EMAIL_PASSWORD=;EMAIL_USERNAME=;SN_ACCOUNT_HOST=;SN_ACCOUNT_PORT=;SN_AGGREGATOR_PORT=;SN_DB_HOST=;SN_DB_NAME=;SN_DB_PASSWORD=;SN_DB_PORT=;SN_DB_USER=;JWT_SIGNATURE=;SN_HOST=
```


# Build and Run

```
mvn clean build
```

```
docker build -t account-service . --load
```

```
docker run -p 8080:8080 -e "EMAIL_FROM=.. EMAIL_PASSWORD=.. EMAIL_USERNAME=.. SN_ACCOUNT_HOST=.. SN_ACCOUNT_PORT=8080 SN_AGGREGATOR_PORT=.. SN_DB_HOST=.. SN_DB_NAME=.. SN_DB_PASSWORD=.. SN_DB_PORT=.. SN_DB_USER=.. JWT_SIGNATURE=.. SN_HOST=.. " --rm account-service
```
